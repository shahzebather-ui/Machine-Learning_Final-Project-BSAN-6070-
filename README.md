# Machine Learning Final Project (BSAN-6070)
Machine Learning Final Project: Region 9 heat-health prediction, model tuning, explainability visuals, and Streamlit app.

This repository contains our BSAN6070 final project on predicting heat-related illness (HRI) rates for HHS Region 9 (CA, AZ, NV, HI) using machine learning. Our current Member 1 model is a Decision Tree Regressor with documented hyperparameter tuning, overfitting checks (train vs test), and before/after performance comparisons. The repo includes the finalized dataset used for modeling, reproducible notebooks/scripts, generated analysis visuals, and a Streamlit app for interactive prediction and demo deployment.
